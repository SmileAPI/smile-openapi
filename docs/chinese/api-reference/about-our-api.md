# 认证

Basic authentication 是 HTTP 协议中的一个简单认证方案。如要使用它，请在发送您的 HTTP 请求时使用授权头，其中包含 Basic 这个单词，后接一个空格和一个 base64 编码的用户名：密码。

> 📘 样例
>
> Authorization: Basic ZGVtbzpwQDU1dzByZA==
